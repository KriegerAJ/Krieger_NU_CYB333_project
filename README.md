repo created for National University CYB333 class project

By running the script contained, you agree that you, the user, are held solely responsible for running the code in an approved environment.

The purpose of this project is to create a simple keylogger that saves every 30 seconds to a file in the same folder the script is in.

DEPENDENCIES:
This script requires the pynput library in order to function properly. Without this library, python is unable to monitor the keyboard and thus cannot record keystrokes.

This program *can* be run at startup and *can* be run in the background.
